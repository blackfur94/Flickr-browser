# Flickr browser

Aplikacja na urządzenia mobilne z systemem Android, pozwalająca na wyświetlanie i wyszukiwanie zdjęć w serwisie Flickr za pomocą tagów.

# Wykorzystane technologie i biblioteki:

- Kotlin
- Android Studio
- Flickr API
